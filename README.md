## Run Locally

Clone the project

```bash
  git clone https://github.com/nareshkumaralaria/interactly-assignment-2.git
```

Go to the project directory

```bash
  cd interactly-assignment-2
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

Open http://127.0.0.1:5173/ to view it in your browser.
